# 副本世界预制数据生成指南

## 📋 概述

本方案实现了副本世界的**背景和角色生成提前预制**，使用 **qwen3_max** 模型进行高质量生成。

### 优化内容

1. **背景介绍预制**：使用 qwen3_max 为每个世界生成固定的开场背景（包含随机选项语法）
2. **角色数据预制**：
   - 种族/职业与外貌描述（t_world_race_appearance）
   - 性别与个性描述（t_world_gender_personality）
   - 角色名字池（t_world_character_names）
3. **性能优化**：
   - 角色生成不再实时调用 LLM 生成名字和状态
   - 名字从预制池随机抽取
   - 状态基于个性关键词简单映射

## 🎯 随机选项语法

生成的背景介绍支持随机选项，格式：`[选项 A|选项 B|选项 C]`

**示例：**
```
你走在 [昏暗的街头 | 细雨蒙蒙的小巷 | 落满灰尘的走廊] 中，
迎面走来了一个叫 [艾德 | 路德维希 | 西蒙] 的男人。
```

运行时，`_process_fixed_intro()` 方法会自动替换这些选项，每次游戏体验都不同！

## 🔍 故障排查

### 问题 1：生成失败
**可能原因：** API 密钥过期或配额不足  
**解决：** 检查 `config/config.yaml` 中的 qwen3_max 配置

### 问题 2：数据库连接失败
**可能原因：** 数据库配置错误或网络问题  
**解决：** 检查 `config/config.yaml` 中的 database 配置

### 问题 3：预制数据未生效
**可能原因：** 缓存或未重启服务  
**解决：** 重启 Flask 服务，清除缓存
